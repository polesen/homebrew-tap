# Homebrew Tap — polesen

Homebrew formulae for [@polesen](https://github.com/polesen)'s tools.

## Install

```bash
brew install polesen/tap/<formula>
```

e.g. `brew install polesen/tap/datacow`. This is shorthand for
`brew tap polesen/tap && brew install <formula>`. Upgrade later with
`brew upgrade <formula>`.

## ⚠️ This repository is auto-generated

The files in `Formula/` are generated and pushed automatically by each tool's
release pipeline in its **source repo** (see the Tools table above) on every
tagged release. **Do not edit them by hand — changes are overwritten on the
next release.**

To report a bug, request a feature, or change how a formula is built, open it
in that tool's source repository (linked above) — not here. Pull requests
against this tap will be closed in favour of changes in the source repo.
